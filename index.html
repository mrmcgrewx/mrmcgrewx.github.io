Works

Applications currently available on my public github.

iOS

App Coordinator - Demo App

An app that uses an app coordinator design pattern to abstract app navigation logic from your view controllers. As great as storyboard segues are, they show their achilles heel when it comes time to upgrade or modfiy your application. With this design pattern you can easily modify and change your view, while maintaining the underlying navigation logic of the app, allowing for faster development times.

MLApp - Concept App

This app builds off of the app coordinator design in the Demo App. Wanting to explore and learn more about the Swift CoreML library, I designed a simple app that scans objects in real time and uses the VGG16 model, provided by Apple, to identify the object on the screen. If there is over a 50% confidence rating returned from the data model, the app will take a picture of the object currently on the screen.

When the image is previewed, the user has the option to send the data to an external server or rescan other objects. If the user decides to send the data, image and location data are transferred to an external API. Check out the JavaServer Spring Boot project for a server to run this application with.

Java

Basic Java Spring Boot Server

A Java RESTful API that provides basic user creation and login operations, as well as data collection. This API compliments the MLApp, collecting image and location data from the MLApp and storing it to a mongodb database.This server is built with maven, and it can also be deployed in docker. Documentation is provided in the repo.
